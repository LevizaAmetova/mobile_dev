# Лабораторная работа №1: Знакомство с Flutter

## Цель работы:
- Освоить начальную настройку среды разработки для Flutter (VS Code или Github Codespaces).
- Познакомиться с процессом создания, запуска и публикации Flutter-проекта.
- Изучить базовые виджеты Flutter (`Container`, `Row`, `Column`, `Text`) и их свойства.
- Научиться создавать простой пользовательский интерфейс с использованием `Scaffold` и интерактивных элементов (`FloatingActionButton`).

## Как запустить проект:
- Убедитесь, что в нижней панели VS Code выбран эмулятор или устройство (`Ctrl+Shift+P` -> "Flutter: Select Device").
- Запустите приложение одним из способов:
- Команда в терминале: `flutter run`
- Кнопка "Play" (`Start Debugging`) над файлом `lib/main.dart`.
